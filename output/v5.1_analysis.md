- 止盈次数虽减少（350→250），但去掉的都是 Tier B 的低效交易

## EV 精算

```
P(止盈)   = 92.9%  → 平均赚 +$2.71
P(结算赢) = 3.3%   → 平均赚 +$3.56
P(硬止损) = 3.7%   → 平均亏 -$44.64

单笔 EV = 0.929 × 2.71 + 0.033 × 3.56 + 0.037 × (-44.64)
        = 2.52 + 0.12 + (-1.65)
        = +$0.98 /笔  ✅
```

**单笔 EV 从 -$0.81 翻正到 +$0.98**。

## 权益曲线特征

1. **2026.01**: 缓慢上升 $10,000 → $10,260（稳定止盈积累）
2. **2026.02 初**: 一次回撤 -$200（集中硬止损），但很快回升

曲线形态比 V5.0 显著改善：回撤更浅、恢复更快。

## 剩余风险

1. **仍依赖高胜率**：87.7% 的胜率需要长期维持，任何市场结构变化（如做市商撤离）都可能打破
2. **样本期仅 2 个月**：未覆盖极端事件（如 2024 美国大选期间的市场波动）
3. **1 小时粒度局限**：实际 Maker 单的成交等待时间无法模拟
4. **无流动性过滤**：回测假设所有市场都有足够深度

## 结论

V5.1 优化有效：**砍掉 Tier B + 收紧硬止损到 0.85** 将策略从亏损 (-3.22%) 翻转为盈利 (+2.64%)。

核心逻辑验证：
- 快消型市场 (<12h) 的尾端套利确实存在正 EV
- 控制单笔最大亏损是盈利的关键（不是提高胜率）
- Maker-Only 手续费优势在小利润空间里至关重要

**下一步**：建议用更长时间跨度（6-12个月）的数据验证稳健性，并加入流动性过滤和价格趋势过滤进一步提升胜率。
